# Cosmera - Beauty Products App

## Overview
Cosmera is a modern beauty products mobile application designed in Figma. The application focuses on providing users with a seamless shopping experience through an elegant interface, intuitive navigation, and visually appealing product displays.

## Features
* Modern UI Design
* Beauty Product Catalog
* Product Search Interface
* User-Friendly Navigation
* Shopping Cart Screens
* Responsive Mobile Layout
* Clean and Aesthetic Design

## Tools Used
* Figma
* UI/UX Design Principles
* Wireframing and Prototyping

## Screens Included
* Splash Screen
* Home Screen
* Product Details Screen
* Categories Screen
* Shopping Cart Screen
* User Profile Screen
